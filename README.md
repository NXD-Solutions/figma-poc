# Figma PoC

PoC for validating Figma as the source of truth for NXD frontend development.

**Live app:** https://nxd-solutions.github.io/figma-poc/

## Development

```bash
npm install
npm run dev
```

## Deployment

The app is automatically built and deployed to GitHub Pages on every push to `main` via GitHub Actions (`.github/workflows/deploy.yml`).

No manual steps required — merge to `main` and the site updates within ~1 minute.
